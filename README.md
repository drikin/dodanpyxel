# DodanPyxel シューティングゲーム

縦スクロールシューティングゲーム（Dodonpachiスタイル）を、Pyxelゲームエンジンを使用してiPhone向けに開発しました。モバイル向けのタッチコントロールに最適化されたブレットヘルスタイルのゲームプレイを提供します。

## 機能

- 垂直スクロールシューターゲームプレイ
- 常時オートシューティング
- 高度なパワーアップシステム
- サイクル式ボスシステム（10種類のユニークなロボットボス）
- ゲームの進行とともに大きさが変化するボス
- モバイル向けタッチコントロール
