DodanPyxel - ダウンロード

以下のプラットフォーム向けのバイナリをダウンロードできます。

Windows版

Windows 10/11で動作します

ダウンロード (.zip)
ダウンロード後、.zipファイルを解凍して
dodanpyxel.batを実行してください。 ※Python環境が自動検出され、必要に応じてPyxelライブラリがインストールされます。

macOS版

macOS 10.14以降で動作します

ダウンロード (.zip)
ダウンロード後、zipを解凍して
dodanpyxel.shを実行してください。 ※Pythonがインストールされている必要があります。

ソースコード

全プラットフォーム向け

ダウンロード (.zip)
Pythonとpyxelライブラリが必要です:
pip install pyxel
解凍後:
python main.py
