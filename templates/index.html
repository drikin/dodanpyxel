DodanPyxel - Vertical Shooter

ゲーム情報

DodanPyxelは、古典的な「弾幕系」シューティングゲームを再現した縦スクロールシューターです。

敵の弾を避けながら、できるだけ多くの敵を撃墜して高スコアを目指しましょう！

キーボード操作

矢印キー: 移動

自動発射機能有効！

スペース: ゲーム開始/リスタート

タッチ操作

タッチ＆ドラッグ: 移動

自動発射機能搭載！

画面タップ: ゲーム開始/リスタート

ゲームをダウンロード

インストールと実行方法

1. Pythonをインストール (3.7以上):

Python公式サイトからダウンロード

2. Pyxelライブラリをインストール:

pip install pyxel

3. ダウンロードしたZIPファイルを解凍

4. ゲームを実行:

python main.py
注意: Pyxelのバージョンによって互換性の問題が発生する場合があります。

新機能：自動発射モードが有効になりました！

  1. 自動発射：プレイヤーは移動に集中できるようになりました
  2. 矢印キーまたはWASDキーを使って移動できます
  3. タッチ操作でもスムーズに動作するよう最適化されています
  4. 発射音が無効化されています

最新版では、より多くのプラットフォームとキーボード配列で動作するよう互換性が向上しています。
